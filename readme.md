# Task States

Plugin para alternar estados de tarefas no Obsidian ao clicar no checkbox da lista.

## Como Funciona

Ao clicar em uma tarefa, o plugin evita a marcação padrão e percorre uma sequência de estados.

Sequência de estados:

- `[ ]` TODO
- `[*]` STAND BY
- `[x]` DONE
- `[-]` CANCELLED
- `[!]` PRIORITY
- `[>]` PROGRESS

Funciona em notas Markdown com listas de tarefas (`- [ ]`, `* [ ]` ou `+ [ ]`).

## Requisitos

- Obsidian desktop com plugins da comunidade habilitados

## Uso

1. Abra uma nota com lista de tarefas.
2. Clique diretamente no checkbox da tarefa.

## Status

Alterna o estado da tarefa na própria linha apenas no modo de edição.
