# Task States

Plugin para gerenciar estados de tarefas no Obsidian.

## Requisitos
- Obsidian Desktop (modo Developer)

## Instalação manual
1. Abra o Obsidian e vá em `Configurações` > `Sobre`.
2. Clique em `Abrir pasta de plugins`.
3. Crie uma pasta chamada `task-states`.
4. Copie os arquivos deste plugin para essa pasta:
   - `manifest.json`
   - `main.js`
   - `styles.css`
5. Volte ao Obsidian e ative o plugin em `Plugins da comunidade`.
